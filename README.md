Git & GitHub
ინსტალაცია
დააკლონეთ რეპოზიტორია:
git clone <repository-url>
შედით პროექტის საქაღალდეში:
cd football-page
დააინსტალირეთ საჭირო პაკეტები:
npm install
შექმენით .env ფაილი და დაამატეთ:
API_FOOTBALL_KEY=YOUR_API_KEY
API_FOOTBALL_TIMEZONE=Asia/Tbilisi
API_FOOTBALL_LEAGUES=1,2,3,39,61,78,88,135,140
გაუშვით პროექტი:
npm start
